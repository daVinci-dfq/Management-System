## 技术栈

采用前后端分离技术，

- 使用 Next.js 做 React 的纯前端（当时纠结了很久是做纯前端还是全栈，考虑到学校里大部人可能都会用传统的 Spring Boot 做后台管理，感觉还是不要冒这个头比较好，刚好感觉 Java 的熟练度还不是很高）
- 后端使用 Spring Boot 框架，JPA 做数据持久化。

个人感觉很主流的打法，网上能够借鉴的大佬的资料比较多，学起来并不难。
